how to install this module

Hi and welcome to this quick tutorial to my small and effective npm fetch module

to install this module go to your node module and type:

npm init and then
npm install @arnebuchter/fetch

Here are some given examples on how to use this module

first of all you have to have the following jquery string i a script tag in your html

src="https://code.jquery.com/jquery-3.4.1.min.js" integrity="sha256-CSXorXvZcTkaix6Yvo6HppcZGetbYMGWSFlBw8HfCJo=" crossorigin="anonymous"

Then you can use it like this

let myFetch = myFetch.init({ address: "https://reqres.in/api/", key: "1234" });

key is mandatory, if the API you are working with, do not need a key - type 1234

myFetch.get("users/2") .then(result => { console.log(result); })

users/2 is an example of the resource from your chosen API

you can use get, post, put, kill(delete)
